## Project description (long)

> Frontier vision AI labs spend eight figures a year buying training data. The data they get is sparse, rights-ambiguous, and biographically illiterate. There is no autonomous purchasing surface for provenance-verified, rights-cleared training assets.
>
> Maestra is that surface. She is a two-agent system (Maestra strategic + Apprentice tactical) running on Cloud Run, signed by Kite Passport, that monitors X for new Aeternum Asset drops, hash-verifies via Aegis registry, orders The Aeternum Sour cocktail at the GCX Cocktail Bar for forensic-reasoning priming, evaluates with a Dual-Consensus Agent Protocol (Forensic Skeptic + Art Historical Architect threads in adversarial consensus), and atomically settles training-rights licenses on Base mainnet.
>
> Three transactions per acquisition, ~$1.02 USDC total — Phase 1 ($0.01 → Aegis hash search), Phase 2 ($0.01 → Cocktail Bar with Kite Passport Happy Hour), Phase 3 ($1.00 split $0.95 artist / $0.05 platform). After settlement, the AO Registrar receives a LicenseGranted Amendment — the asset's biographical chain extended; the artist retains ownership; the lab earns non-exclusive training rights, recorded forever.
>
> The GCX Cocktail Bar is novel agent infrastructure: agents pay $0.01–$0.50 micropayments for dense-context payloads (10K–80K tokens) that scaffold their reasoning grammar for a single task session — no fine-tuning, no weight updates, no catastrophic forgetting. Trust as math, not as text. Framework paper: doi.org/10.5281/zenodo.18667742.
>
> Bonus criteria hit by default: multi-agent coordination (Maestra + Apprentice + Bar + Aegis), gas abstraction (Kite Passport two-wallet routing transparently bridges Kite chain → Base mainnet), scoped permissions (Passport ephemeral session keys cap spending per cycle, auto-expire, are revocable). Cloud Run + Firebase Hosting + Vertex/Gemini = full Google stack, MIT-licensed, `docker-compose up` reproduces locally with stub mode for cold-start judges.
>
> Live proof: Kite Passport → x402 → Base mainnet validated 2026-04-29 with tx 0x09deffc1…7623 ($0.01 weather call) and 0xa8c7f3fc…840886 (GCX Bar smoke test). The bridge to Arweave/AO is the post-Buildathon protocol commitment: A2A × x402 Permaweb Extension SDK — middleware so any Kite-attested agent can post AO Amendments without holding AR.
